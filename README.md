## Build (PS Vita)

Install VITASDK (set `VITASDK` env var) and SDL2 for Vita.

```
make vita
```
Artifacts (.vpk) end up in `build/vita`.

## Testbuild (PC)

Dependencies: SDL2 development libraries.

```
make pc
./build/pc/gravity_hunters
```
